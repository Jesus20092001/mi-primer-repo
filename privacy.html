3. Uso de Cookies y Almacenamiento Local

Este sitio utiliza únicamente almacenamiento local (localStorage) para dos funciones puramente técnicas:

  • Preferencia de tema: Guarda si el usuario prefiere modo oscuro o claro.
  • Preferencia de idioma: Guarda si el usuario prefiere español o inglés.

Estos datos se almacenan exclusivamente en tu navegador, no se transmiten a ningún servidor, no se comparten con terceros y puedes eliminarlos en cualquier momento desde la configuración de tu navegador.

4. Compartir Información con Terceros

Los datos personales no se venden, alquilan ni comparten con terceros para fines de marketing o publicidad. Únicamente se pueden compartir datos en las siguientes circunstancias limitadas:

  • Proveedores de hosting: Los datos del sitio se alojan en servidores de terceros (GitHub Pages, Vercel, Netlify) que operan bajo sus propias políticas de privacidad.
  • Obligaciones legales: Si una autoridad competente lo requiere mediante orden judicial válida.
  • Con consentimiento: Si el usuario autoriza explícitamente compartir información específica.

Aplicamos las siguientes medidas de seguridad para proteger la información:

  • Comunicación por email y WhatsApp con cifrado TLS/SSL.

  • Eliminación segura de datos de proyectos finalizados tras el período de garantía.
  • Acceso restringido: solo el desarrollador tiene acceso a los datos de clientes.
  • Repositorios privados para código de clientes; nunca se publica código propietario sin autorización.

  • Oposición: Oponerte al tratamiento de tus datos en determinadas circunstancias.
  • Portabilidad: Recibir tus datos en un formato estructurado y de uso común.
